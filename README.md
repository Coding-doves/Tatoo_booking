# Legit Ink Tattoo Booking Website

## Description

This is a static website for Legit Ink Tattoo Nigeria, a professional tattoo studio offering custom tattoos, tattoo removal, piercing services, and aftercare products. The website provides an easy way for customers to book appointments, explore services, and contact the studio. Built with modern web technologies, it features a responsive design that works seamlessly on desktop and mobile devices.

## Features

- **Responsive Navigation**: Mobile-friendly menu with toggle functionality.
- **Image Slider**: Automatic and manual slideshow showcasing studio highlights.
- **Booking Forms**: Multiple forms for booking appointments via email or WhatsApp.
- **Services Section**: Detailed overview of services including Custom Tattoos, Tattoo Removal, Piercing, and Aftercare Products.
- **Contact Information**: Direct contact options including phone, email, and social media links.
- **Opening Hours**: Display of studio operating hours.
- **About Section**: Information about the studio and its commitment to quality.
- **Footer**: Comprehensive footer with contact details, social links, and credits.

## Technologies Used

- **HTML5**: Structure and content of the website.
- **CSS3**: Styling, animations, and responsive design with custom CSS variables.
- **JavaScript**: Interactive elements including navigation toggle and image slider.

## Setup Instructions

1. Clone or download the project files to your local machine.
2. Ensure you have a modern web browser installed (e.g., Chrome, Firefox, Safari).
3. Open the `index.html` file directly in your web browser to view the website.

No additional setup or server is required as this is a static website.

## Usage

- **Navigation**: Use the top navigation menu to jump to different sections like Home, About, Services, and Contact.
- **Booking**: Fill out the booking forms in the "Book an Appointment" sections. You can book via email or directly via WhatsApp.
- **Services**: Explore the services offered and click "Book Now" to proceed with booking.
- **Contact**: Use the contact form or provided phone/email for inquiries.
- **Mobile**: On mobile devices, tap the menu icon to access the navigation.
